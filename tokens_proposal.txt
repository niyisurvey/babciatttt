TOKEN GROUPING PROPOSAL (implementation-agnostic)

1) Palette
- palette.primary: brand primary interactive color.
- palette.onPrimary: content on primary surfaces.
- palette.secondary: secondary brand accent.
- palette.onSecondary: content on secondary surfaces.
- palette.tertiary: tertiary accent / decorative.
- palette.success: positive status / completion.
- palette.warning: caution / pending status.
- palette.error: destructive / failure state.
- palette.neutral.0/50/100/200/300/400/500/600/700/800/900: grayscale scale.
- palette.text.primary: primary readable text.
- palette.text.secondary: secondary readable text.
- palette.text.inverse: text on dark/primary surfaces.
- palette.border.subtle/strong: border colors.
- palette.icon.primary/secondary/status: icon colors.

2) Backgrounds
- backgrounds.main: default app background.
- backgrounds.sheet: modal sheet background.
- backgrounds.card: card background.
- backgrounds.overlay.dim: modal/overlay dim.
- backgrounds.overlay.scrimStrong: stronger scrim.
- backgrounds.gradient.primary: primary branded gradient.
- backgrounds.gradient.weather.sunrise/day/sunset/night: specialized gradients.
- backgrounds.gradient.areas: areas screen gradient.

3) Surfaces
- surfaces.card.default: default card fill.
- surfaces.card.elevated: elevated card fill.
- surfaces.toast.info/success/warning/error: toast backgrounds.
- surfaces.tooltip: tooltip background.
- surfaces.control.default/pressed/disabled: control fills.

4) Elevation
- elevation.shadow.color: shadow color token.
- elevation.shadow.opacity.level1/level2/level3: shadow alpha scale.
- elevation.shadow.radius.level1/level2/level3: shadow radius scale.
- elevation.shadow.offsetY.level1/level2/level3: shadow offsets.
- elevation.overlay.dim/subtle/strong: overlay opacity scale.
- elevation.level1/level2/level3: composite elevation presets (color+opacity+radius+offset).
- elevation.glass.tintOpacity: glass tint alpha.
- elevation.glass.glow.high/low: glow strengths.

5) Grid (spacing + sizing scale)
- grid.xxxl/xxl/xl/l/m/s/xs/xxs: canonical spacing scale.
- grid.sectionSpacing: default vertical group spacing.
- grid.listSpacing: list row spacing.
- grid.cardPadding/default/tight: padding presets.
- grid.button.paddingX/paddingY: button internal padding.
- grid.button.minHeight: minimum tap height.
- grid.icon.tiny/small/medium/large/xl/xxl: icon sizes.
- grid.dimension.heroCardWidth/heroCardHeight: key layout dimensions.
- grid.offset.subtle/medium/strong: offsets for motion/layout.

6) Shape
- shape.hero: large hero radius.
- shape.card: standard card radius.
- shape.control: button/control radius.
- shape.tooltip: tooltip radius.
- shape.subtle: small radius accents.
- shape.round: fully rounded / capsule usage.
- shape.border.width.default: default border width.
- shape.border.opacity.subtle/strong: border opacities.

7) Typography
- typography.largeTitle: large hero titles.
- typography.title: primary section titles.
- typography.title2: secondary titles.
- typography.headline: emphasized body headings.
- typography.body: default body copy.
- typography.bodyEmphasis: emphasized body copy.
- typography.caption: supporting text.
- typography.label: control labels.
- typography.metric: numbers, counters, points.

8) Motion
- motion.duration.fast/normal/slow: timing scale.
- motion.spring.list: list animation spring.
- motion.spring.hero: hero transitions.
- motion.curve.standard/emphasized: motion curves.

9) Icon sizes and strokes
- icon.size.tiny/small/medium/large/xl: semantic icon sizes.
- icon.stroke.subtle/standard/strong: stroke widths.

NOTES ON EXISTING TOKEN STRUCTURE OBSERVED
- The repository already defines a Theme structure with palette, gradients, typography, motion, shape, grid, glass, and elevation groups.
- This proposal aligns with the existing grouping while standardizing naming to semantic, scale-based tokens.
