Palette
- palette.primary: Primary brand/action color.
- palette.onPrimary: Content on primary surfaces.
- palette.secondary: Secondary brand accent.
- palette.tertiary: Support accent.
- palette.success: Positive/status success.
- palette.warning: Caution/status warning.
- palette.error: Destructive/status error.
- palette.neutral: Neutral ink/shadow base.
- palette.textPrimary: Default text color.
- palette.textSecondary: Secondary/deemphasized text.
- palette.textInverse: Text on dark/brand surfaces.

Backgrounds
- backgrounds.main: App root background.
- backgrounds.sheet: Modal/sheet background.
- backgrounds.card: Card base background.
- backgrounds.overlay: Scrims/dims.
- backgrounds.gradient.primary: Primary gradient array.
- backgrounds.gradient.areas: Areas gradient array.
- backgrounds.gradient.weather: Weather gradient array.

Surfaces
- surfaces.card: Elevated card surface.
- surfaces.toast: Toast/banner surface.
- surfaces.tooltip: Tooltip/help surface.
- surfaces.glass: Glassmorphism base surface.

Elevation
- elevation.shadowColor: Base shadow color.
- elevation.overlayDim: Standard dim opacity.
- elevation.level0.opacity/radius/y: Resting elevation.
- elevation.level1.opacity/radius/y: Card elevation.
- elevation.level2.opacity/radius/y: Modal elevation.

Grid
- grid.xxl/xl/l/m/s/xs: Spacing scale.
- grid.cardPadding: Default card padding.
- grid.cardPaddingTight: Compact card padding.
- grid.listSpacing: Spacing between list items.
- grid.sectionSpacing: Spacing between sections.
- grid.buttonPaddingX/Y: Button interior padding.
- grid.icon.*: Semantic icon sizes.

Shape
- shape.hero: Large hero radius.
- shape.card: Card radius.
- shape.control: Control radius.
- shape.tooltip: Tooltip radius.
- shape.subtle: Small radius.
- shape.borderWidth: Standard border width.

Typography
- typography.largeTitle: Large hero headings.

Motion and icon sizes
